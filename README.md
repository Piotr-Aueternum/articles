# Article list project
Simple Article list

# Start
  Preferred environment `node.js@7.x` and `npm@4.x`
  ```
  npm install
  ```

# Development
  If you want to run this on LAN host, please set up data in config(.env) `LAN` to your workstation IPv4(`xxx.xxx.xxx.xxx`).
  ```
  npm run build:dev
  ```

# Production
  ```
  npm run build:dist
  ```
